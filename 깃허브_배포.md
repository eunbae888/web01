1. 깃설정
git init

2. 깃허브 환경설정 아이디(유저이름)
git config user.name 'enubae08'

확인 유저이름등옥
git config user.name

3. 깃허브 이메일
git config user.email 'enubae08'

확인 메일등옥
git config user.email

4. 배포할 리포지토리 오리진 등록
git 원격지접속 추가  오리진 https://github.com/eunbae888/web02.git
git remote add origin 'https://github.com/eunbae888/web02.git'

확인 검증
git remote -v

5. 모든 파일(.) 추가(모든파일 선택)
git add .

6. 모든 선택된  파일의 메시지 등록
git commit -m '웹페이지배포'

7. 모든 선택된 파일의 마스터에게 올리기 
git push 오리진(배포주소)에 주인에게 보낸다
git push origin master

8. 모두 푸쉬 완료

9. 깃허브 브라우저에서 새로고침 => 올린파일 확인

10. 셋팅(settings) 메뉴 클릭

11. 좌측 중간 하단에 => Pages 클릭

12. 우측에 None를 클릭 => master 선택

13. Save 클릭

14. 1분정도 대기하고 새로고침 하면 배포주소 보인다. => Visit Site 클릭 홈페이지 뜬다
